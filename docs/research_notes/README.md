# Research Notes

Use this directory to store lab notes, experiment logs, and incremental findings.

## Suggested Structure

```
research_notes/
├── README.md                  ← this file
├── experiment_log.md          ← running log of experiments and results
├── literature_notes.md        ← summaries of relevant papers
└── data_observations.md       ← notes on dataset characteristics
```

## Guidelines

- Date all entries (`## YYYY-MM-DD`)
- Record both successes and failures
- Note any anomalies observed in the data or model outputs
- Link to relevant notebook cells where applicable
